# Investigación — Plataforma AIoT Distribuida de Percepción Visual

## Descripción General

La carpeta `b_investigacion/` contiene el cuerpo completo de investigación técnica, conceptual y arquitectónica del proyecto.

Su propósito consiste en documentar, analizar y validar una arquitectura AIoT distribuida basada en percepción visual, estructurada sobre un modelo Edge–Fog–Cloud orientado a la gestión inteligente de la información.

A diferencia de un enfoque centrado únicamente en implementación tecnológica, esta investigación aborda simultáneamente:

- fundamentos teóricos;
- arquitectura distribuida;
- percepción computacional;
- inteligencia artificial;
- telecomputaciones;
- resiliencia;
- observabilidad;
- automatización operacional;
- gestión informacional AIoT.

La investigación fue desarrollada como parte del proceso de construcción conceptual y técnica del proyecto, funcionando tanto como soporte del trabajo práctico como base estructural para una futura tesina de grado.

---

# Objetivo de la Investigación

El objetivo principal de esta investigación consiste en estudiar y estructurar una plataforma AIoT distribuida capaz de transformar datos visuales en eventos operacionales inteligentes mediante procesamiento distribuido entre capas Edge, Fog y Cloud.

La investigación busca responder problemáticas centrales relacionadas con:

- centralización excesiva del procesamiento;
- latencia operacional;
- desacople entre capas computacionales;
- escalabilidad de sistemas distribuidos;
- resiliencia frente a fallos;
- distribución de inteligencia artificial;
- automatización de la gestión de información.

---

# Hipótesis Central

La investigación parte de una hipótesis fundamental:

> Los sistemas AIoT modernos requieren arquitecturas distribuidas capaces de gestionar información de manera inteligente mediante separación funcional entre percepción, procesamiento, análisis y presentación a través de múltiples capas computacionales.

Bajo esta visión:

- Edge;
- Fog;
- Cloud;

no representan únicamente ubicaciones físicas de procesamiento, sino regiones funcionales dentro de una infraestructura informacional distribuida.
