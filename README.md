# Descripción General

Este repositorio contiene el desarrollo completo del trabajo de investigación:

> **“Sistemas de Visión por Computador Distribuida para Gestión y Control basados en Arquitecturas AIoT”**

realizado en el marco de la materia **Visión Artificial** perteneciente a la carrera:

- Licenciatura en Automatización y Control  
- Universidad Tecnológica Nacional — Facultad Regional Córdoba (UTN-FRC)

El proyecto investiga cómo la visión por computador puede integrarse dentro de arquitecturas distribuidas basadas en modelos:

- Edge;
- Fog;
- Cloud;

para transformar percepción visual en información operacional estructurada capaz de participar en procesos de:

- monitoreo;
- automatización;
- análisis;
- coordinación;
- actuación distribuida;
- control operacional.

---

# Objetivo de la Investigación

La investigación busca analizar y estructurar una arquitectura AIoT distribuida capaz de procesar información visual en tiempo real mediante distribución funcional del procesamiento entre múltiples capas computacionales.

El trabajo aborda problemáticas relacionadas con:

- latencia operacional;
- procesamiento distribuido;
- inferencia visual en Edge;
- resiliencia operacional;
- seguridad transversal;
- escalabilidad;
- observabilidad;
- coordinación entre capas Edge–Fog–Cloud.

---

# Hipótesis Central

La investigación parte de la siguiente hipótesis:

> Los sistemas modernos de percepción visual requieren arquitecturas distribuidas capaces de desacoplar percepción, procesamiento, análisis y actuación mediante múltiples niveles computacionales coordinados.

Bajo esta visión:

- Edge;
- Fog;
- Cloud;

son interpretados como regiones funcionales dentro de una infraestructura informacional distribuida.

---

# Ejes Fundamentales del Proyecto

La investigación se estructura alrededor de cuatro ejes principales:

- poder de cómputo;
- latencia;
- inteligencia artificial;
- seguridad.

Estos ejes son analizados transversalmente sobre toda la arquitectura propuesta.

# Arquitectura General

La infraestructura propuesta se organiza mediante un modelo distribuido:

```text
[ EDGE ]
Percepción · Inferencia · Captura · Actuación inmediata

        ↓

[ FOG ]
Coordinación · Correlación · Eventos · Decisiones operacionales

        ↓

[ CLOUD ]
Analítica · Entrenamiento · Supervisión · Escalabilidad
```

---

# Dominios de Aplicación

La arquitectura investigada se analiza sobre múltiples dominios operacionales:

- automatización industrial;
- control de calidad;
- gestión de tráfico;
- seguridad vial;
- monitoreo ambiental;
- gestión de emergencias;
- agricultura de precisión.

La infraestructura base permanece constante entre dominios, modificándose únicamente:

- modelos de IA;
- reglas operacionales;
- actuadores asociados.

# Consideraciones Finales

Este repositorio documenta una investigación orientada al estudio de sistemas de percepción visual distribuidos aplicados a arquitecturas AIoT modernas.

El proyecto propone una visión estructural donde la percepción computacional deja de ser interpretada como un proceso aislado y pasa a integrarse dentro de infraestructuras distribuidas capaces de:

- percibir;
- procesar;
- analizar;
- coordinar;
- comunicar;
- y actuar;

sobre entornos físicos mediante inteligencia artificial y procesamiento distribuido.
